How It Works

If an item is not available at Troy Public Library or through the Upper Hudson Library System catalog, you may be able to request it through Interlibrary Loan.

Request Options

You can submit a request in person at the Information Desk or through the library’s online request form.

Who Can Use This Service

Interlibrary Loan is available to Troy Public Library cardholders in good standing.

What to Expect

The lending period is set by the library that owns the item, not by Troy Public Library.

Renewals are only possible if the lending library allows them.

Fees & Limits

Request limit Up to 3 items per week
Possible postage fee $2.50 per item
Late fine $0.10/day, max $5.00

Please note

Some materials may be in-library use only, depending on the lending library’s restrictions.

Microfilm is limited to in-library use only.

Need help?

Library staff can help you figure out whether an item can be borrowed through Interlibrary Loan and how to submit a request.
